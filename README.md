# State Capitols Projects

### Part 1
The objective is to create a Python program that will create a valid JSON file that contains all of
the State Capital addresses.

### Part 2
Create a Python program that will read the file from part 1 and produce a file that includes
longitude and latitude for each address.


### File Descriptions
"Addresses JSON.py" is the first script, it reads through the "capitol-buildings-by-state-2026.csv", and it creates the first JSON file "state_capitols.json"

"long_lat state capitols.py" is the second script, getting all the coordinates for each capitol, and creating "state_capitols_with_coords.json", which includes all the information from the first JSON, but with the coordinates.
